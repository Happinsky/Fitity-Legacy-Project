# Fitity - C++ Desktop Application



### Opis 

[PL] Projekt stworzony na zaliczenie w 2023 roku. Jest to aplikacja desktopowa zbudowana w **C++** przy użyciu **Windows Forms (.NET)**. 





### Kluczowe informacje:

* **Status:** Projekt archiwalny.

* **Cel:** Nauka budowania interfejsów graficznych (UI) oraz integracji z zewnętrznymi usługami chmurowymi.

* **Główne funkcje:** System logowania, moduły obliczeniowe oraz eksperymentalna integracja z **Azure Cognitive Services** i **Google Vision API**.



**Uwaga:** Ze względu na to, że jest to projekt studencki sprzed kilku lat, nie rozwijam go już i nie pamiętam szczegółów implementacyjnych każdej funkcji. Ten kod służy jedynie jako dokumentacja moich wczesnych doświadczeń z programowaniem obiektowym i pracą z dokumentacją techniczną.





### Narzędzia / Tools:

- Visual Studio

- C++ / CLI

- .NET Framework (Windows Forms)
